Dates






new Date()

Date is a global object for dealing with dates. Internally, Javascript stores dates in Unix epoch time - the number of milliseconds elapsed since January 01, 1970, 00:00:00 UTC

const currentDateTime = new Date();

Constructors

new Date('May 10, 2021 18:01:02');
new Date('2021-05-10T18:01:02');
new Date(2021, 4, 17);
new Date(2021, 4, 17, 18, 01, 02);
new Date(1620582923000);

Static Methods

  1. Date.now() - Returns current time in epoch millisecond format
  2. Date.parse() - Parses a string and returns epoch millisecond time (not recommended)
  3. Date.UTC() - Similar to the Date() constructor, but in UTC

Instance Methods

  1. get...() : date, day, year, hours, minutes, etc
  2. set...() : date, day, year, hours, minutes, etc
  3. toDateString() - Returns the year/month/date in string format
  4. toISOString() - Returns the date in ISO8601 format (best format)
  5. toLocale*()
  6. toString()
  7. toTimeString()
  8. toUTCString()

Date Comparisons

You can use the "greater than" and "less than" operators to compare dates accurately, but not "===". Use getTime() with the equality operator for that.

const date1 = new Date();
const date2 = new Date();
date2 > date1;
date1.getTime() === date2.getTime();

Manipulating Dates

Javascript doesn't give us much in the way of manipulating dates but there are, of course, third party libraries.

  1. Moment.js (deprecatd)
  2. Luxon
  3. Date-fns

Date Formats (ISO8601)

ISO8601 is the one date format to rule them all. It sorts well on computers, it's an international standard, and it just plain makes sense.

  1. 11:06 05/10/2021
  2. 10/5/21
  3. 15-MAY-2021
  4. 2021-05-11T21:24:51.763Z
