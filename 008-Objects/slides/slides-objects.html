Objects



Objects allow us to create containers to hold other properties (string, numbers, objects, functions, etc) and logically separate them in a hierarchal manner. They are usually used to represent a logical "thing" like a car, pet, person, etc.

Object Instantiation

Objects are declared similarly to other types but members are wrapped in curly braces and assigned in key-value pairs if created at the time of declaration. 
const myObject = {
    myString: 'test string',
    myNumber: 42,
};

Object Member Access

Object members can be accessed and assigned using dot notation or bracket notation. Dot notation is the most common and easiest to read.

// Declare and initialize myObject with two properties
const myObject = {
    myString: 'test string',
    myNumber: 42,
};
// Access and print the value of myString and myNumber
console.log(`myString = ${myObject.myString} and myNumber = ${myObject.myNumber}`);

// Create a new property named "newBoolean" on "myObject" and assign it "true"
myObject.newBoolean = true;
console.log(`The value of myObject.newBoolean is : ${myObject.newBoolean}`);

Object Member Access

Bracket notation is very similar. It uses square brackets after the object with the name of the property you wish to access.

// Declare and initialize myObject with two properties
const myObject = {
    myString: 'test string',
    myNumber: 42,
};
// Access and print the value of myString and myNumber
console.log(`myString = ${myObject['myString']} and myNumber = ${myObject['myNumber']}`);

// Create a new property named "newBoolean" on "myObject" and assign it "true"
myObject['newBoolean'] = true;
console.log(`The value of myObject['newBoolean'] is : ${myObject['newBoolean']}`);

Template Literal Object Member Access

Bracket notation becomes extremely powerful when combined with template literals.

// Declare and initialize "myObject"
const myObject = {
    myString: 'test string',
    myNumber: 42,
};

// Access and print the value of myString and myNumber
console.log(`myString = ${myObject[`${'my' + 'String'}`]} and myNumber = ${myObject[`${'m' + 'yNumber'}`]}`);

// Create new boolean member on myObject named "newBoolean" and assign Boolean "true"
const newBooleanVarName = 'newBoolean';
myObject[`${newBooleanVarName}`] = true;
console.log(`The value of myObject[newBooleanVarName] is : ${myObject[newBooleanVarName]}`);

delete'ing Properties

Object properties can be removed with the delete keyword.

// Declare and initialize "myObject"
const myObject = {
    myString: 'test string',
    myNumber: 42,
};
console.log(myObject.myString); // 'test string'
delete myObject.myString; // delete the "myString" property
console.log(myObject.myString); // undefined

Common Object Static Methods

These are not all of the Object static methods, just the most commonly used

  1. Object.assign(obj1, obj2, ...) - Shallow copy / merge object properties
  2. Object.keys(obj1) - Return an array of obj1 property names (keys)
  3. Object.values(obj1) - Return an array of obj1 values
  4. Object.entries(obj1) - Return an array of arrays containing obj1 key and value

Object.keys() vs for...in

const truck = {
    make: 'Toyota',
    model: 'Tundra',
    year: 2010,
};

Object.keys(truck).forEach(function (key) { // old style
    console.log(`Key: ${key}, Value: ${truck[key]}`);
});

Object.keys(truck).forEach((key) => { // new style / arrow function
    console.log(`Key: ${key}, Value: ${truck[key]}`);
});

for (const key in truck) {
    console.log(`Key: ${key}, Value: ${truck[key]}`);
}

Object.assign() Example

const petDetails1 = {
    name: 'Rin Tin Tin',
    breed: 'German Shepherd',
    firstMovie: {
        name: 'The Man from Hells River',
        year: 1922,
    },
};

const petDetails2 = { type: 'dog', isGoodBoi: true };
const pet = Object.assign({}, petDetails1, petDetails2);
    
console.log(`The ${pet.type}'s name is ${pet.name} and starred in their first movie in ${pet.firstMovie.year}.  Where they a good boi?  ${pet.isGoodBoi}`);

Shallow Copy Warning

const person = {
    name: 'Patti',
    age: 30,
    hobbies: {
        music: true,
        singing: true,
    },
}

const personClone = Object.assign({}, person);
person.hobbies.hurdyGurdy = true;

console.log(personClone.hobbies); // { music: true, singing: true, hurdyGurdy: true }

Object Destructuring

Objects can be "destructured" into their components with automatic property name matching.

const truck = {
    make: 'Toyota',
    model: 'Tundra',
    year: 2010,
};

const { make, model, year } = truck;
