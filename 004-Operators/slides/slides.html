Mathematical Operators

  • Addition : 1 + 2
  • Subtraction : 2 - 1
  • Multiplication : 2 * 2
  • Division : 10 / 5
  • Exponentiation : 5 ** 2
  • Modulus : 9 % 2
  • Increment : x++
  • Decrement : y--
  • Grouping : (1 + 2) * 3

Order of Operations

Operators are applied in this order, then from left to right :

  1. Grouping
  2. Increment, Decrement
  3. Exponentiation
  4. Multiplication, Division, Modulus
  5. Addition, Subtraction

Prefix Increment/Decrement

When used prefix, the operator will apply the change and return the value after it was applied.


            let x = 1;
            console.log(++x); // '2'
            console.log(++x); // '3'

Postfix Increment/Decrement

When used postfix, the operator will apply the change and return the value before it was applied.


            let x = 1;
            console.log(x++); // '1'
            console.log(x++); // '2'

Type Operators

  • typeof : Returns the type of a variable
  • instanceof : Returns true if variable is an instance of the specified type, otherwise false. Note that primitives are... "different"

            const x = { test: true, }
            console.log(typeof(x)); // object
            console.log(x instanceof Object); // true

            const y = 'a string';
            console.log(typeof(y));  // string
            console.log(y instanceof String);  // false.  Wait, false?
            console.log(typeof(y) === 'string'); // true
