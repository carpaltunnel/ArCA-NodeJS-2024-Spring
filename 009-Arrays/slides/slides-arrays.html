Arrays



Arrays allow us to create list-like objects of variables of any type. Arrays are zero indexed so the first item in an array is index 0


Note that the language does not enforce a type for items in the array. While generally a bad practice, it is possible to have a single array that contains strings, numbers, objects, and even other arrays.

Array Instantiation

Arrays are declared similarly to other variables but members are wrapped in square brackets and assigned comma delimited if created at the time of declaration. 
const fruit = [
    'apple',
    'orange',
    'kiwi',
];

const numbers = [
   0,
   1,
   2,
];

Array Access

Array members can be accessed and changed by their index number with bracket notation. Keep in mind that Arrays are zero indexed. The length returns the number of items.

const fruit = [
    'apple',
    'orange',
    'kiwi'
];
fruit.length; // 3
fruit[0]; // apple
fruit[2]; // kiwi

fruit[2] = 'cantaloupe';
fruit[2]; // cantaloupe

Array Static Methods

The most important static method to know for now is Array.isArray(). This allows you test an unknown or uncertain variable to determine if it is an array.

const fruit = [
    'apple',
    'orange',
    'kiwi'
];

const cars = 'are transportation';
Array.isArray(fruit); // true
Array.isArray(cars); // false

Modifying Arrays

The most common methods for adding and removing items from an array are push(), pop(), shift(), and unshift()

  1. arr.push(val) - Add (push) val to end of the arr and return new length.
  2. arr.pop() - Remove (pop) and return the last element of arr.
  3. arr.unshift(val) - Add (unshift) val to the beginning of arr and return new length.
  4. arr.shift() - Remove (shift) and return the first element of arr.
  5. arr.splice(idx, deleteCount, newItem) - Add or delete items of arr at index.

Modifying Arrays - Example

const fruit = ['apple'];
fruit.push('orange'); // 2, fruit = [apple, orange]
fruit.unshift('mango'); // 3, fruit = [mango, apple, orange]
fruit.length; // 3

const lastElement = fruit.pop(); // 'orange', fruit = [mango, apple]
console.log(`fruit no longer contains ${lastElement}`);

const firstElement = fruit.shift(); // 'mango', fruit = [apple]
console.log(`fruit no longer contains ${firstElement}`);
fruit.length; // 1

fruit.pop(); // 'apple', fruit is now an empty array
fruit.pop(); // undefined
fruit.length; // 0

Iterating Arrays

const fruits = ['apple', 'orange', 'kiwi'];

fruits.forEach((fruit) => {
    console.log(`Fruit value is : ${fruit}`);
});

for (const idx in fruits) {
    console.log(`Fruit value is : ${fruits[idx]}`);
}

for (const fruit of fruits) {
    console.log(`Fruit value is : ${fruit}`);
}

Filtering Arrays

const fruits = [ 'apple', 'orange', 'kiwi', 'apricot' ];
const fruitsThatStartWithA = fruits.filter((fruit) => {
    if (fruit.slice(0, 1) === 'a') {
        return true;
    }
});

// Shorthand way of doing the same thing :
const fruitsThatStartWithA2 = fruits.filter(fruit => (fruit.slice(0, 1) === 'a'));

Evaluating Array Contents

.includes(), .some(), .every() let us test the contents of an Array and return a Boolean.


const fruits = ['apple', 'orange', 'kiwi'];

const includesKiwi = fruits.includes('kiwi');
const someContainA = fruits.some(fruit => (fruit.indexOf('a') > -1));
const allContainA = fruits.every(fruit => (fruit.indexOf('a') > -1));

Searching Arrays

.indexOf(), .lastIndexOf(), .find(), .findIndex() search an array.


const fruits = ['apple', 'orange', 'kiwi', 'apple'];
const appleFirstIndex = fruits.indexOf('apple');
const appleLastIndex = fruits.lastIndexOf('apple');
const aFruit = fruits.find(fruit => (fruit.indexOf('a') > -1));
const aFruitIndex = fruits.findIndex(fruit => (fruit.indexOf('a') > -1));

Sorting

.sort() will in place sort an Array by default implicit String conversion or by a supplied function.


const fruits = ['apple', 'orange', 'kiwi', 'apricot'];

fruits.sort(); // default alpha sort
fruits.sort((a, b) => a.length > b.length ? 1 : -1); // length sort

Array Destructuring and Rest

Arrays can be "destructured" for assignment and the ... rest operator can be used for leftovers


const fruits = [
    'apple', 
    'orange', 
    'kiwi', 
    'apricot'
];

const [fruit1, fruit2, ...others] = fruits;
