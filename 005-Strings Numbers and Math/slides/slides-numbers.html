Numbers and Math




Static Methods

  1. Number.parseInt(string) - Parse an integer Number from a String input
  2. Number.parseFloat(string) - Parse a float Number from a String input
  3. Number.isNaN(value) - Check if value is NaN
  4. Number.isFinite(value) - Check if value is finite
  5. Number.isInteger(value) - Check if value is an integer
  6. Number.isSafeInteger(value) - Check if value is a safe integer that can be represented

Instance Methods

  1. .toExponential() - Return a string value in exponential notation
  2. .toFixed() - Returns a string representing the number in fixed point notation
  3. .toLocaleString() - Returns a locale specific string representation of the number
  4. .toPrecision() - Returns a string representation of the number in the specified precision
  5. .toString() - Returns a string representation of the number in the specified base (default 10)
  6. .valueOf() - Returns the primitive value of the object

Math

The Math object is static so all methods and properties can be used without creating an instance. When using round(), ceil(), and floor(), keep in mind negative numbers.

  1. Math.round(value) - Rounds to the nearest integer
  2. Math.ceil(value) - Rounds up to the nearest integer
  3. Math.floor(value) - Rounds down to the nearest integer
  4. Math.trunc(value) - Returns the integer part of the number
  5. Math.random() - Returns a pseudorandom number between 0 (inclusive) and 1 (exclusive)

Examples

const floatingNumber = 27.123;
floatingNumber.toExponential(); // "2.7123e+1"
floatingNumber.toExponential(2); // "2.71e+1"
floatingNumber.toFixed(); // "27"
floatingNumber.toFixed(4); // "27.1230"
floatingNumber.toPrecision(3); // "27.1"
floatingNumber.toPrecision(8); // "27.123000"
Math.round(floatingNumber); // 27
Math.ceil(floatingNumber); // 28
Math.floor(floatingNumber); // 27
Math.trunc(floatingNumber); // 27

const negativeNumber = -43.6; // -43.6
Math.round(negativeNumber); // -44
Math.ceil(negativeNumber); // -43
Math.floor(negativeNumber); // -44
