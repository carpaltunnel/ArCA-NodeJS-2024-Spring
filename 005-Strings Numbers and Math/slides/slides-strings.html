Strings




Template Literals (Template Strings / String Interpolation)




console.log(`1 + 1 = ${1 + 1}`); // 1 + 1 = 2

const theAnswer = 2;
console.log(`The answer to 1 + 1 is : ${theAnswer}`); // The answer to 1 + 1 is : 2

const name = 'user';
console.log(`Hello ${name}, the answer to 1 + 1 is ${theAnswer}, and 2 * 3 = ${2 * 3}`); // Hello user, the answer to 1 + 1 is 2, and 2 * 3 = 6

length, indexOf(), lastIndexOf()




const myString = 'a regular string!';
console.log(myString.length); // 17
console.log(myString.indexOf('r')); // 2
console.log(myString.lastIndexOf('r')); // 12
console.log(myString.indexOf('lar')); // 6
console.log(myString.lastIndexOf('lar')); // 6

toLowerCase(), toUpperCase()



const myString = 'The quick Red Fox...';
console.log(myString.toLowerCase()); // the quick red fox...
console.log(myString.toLocaleLowerCase()); // the quick red fox...
console.log(myString.toUpperCase()); // THE QUICK RED FOX...
console.log(myString.toLocaleUpperCase()); // THE QUICK RED FOX...

The "Locale" versions convert based on the locale and therefore may perform differently based on the language or detected/set locale.

substr(), substring(), slice()

substr(start, length) can be considered legacy. substring(start, end) is most common. slice(start, end) is a bit uncommon but combines the best of both.

const myString = 'The quick Red Fox...';
console.log(myString.substr(4, 5)); // quick
console.log(myString.substr(-6, 6)); // Fox...

console.log(myString.substring(4, 5)); // q
console.log(myString.substring(-6, 6)); // The qu
console.log(myString.substring(4)); // quick Red Fox...

console.log(myString.slice(4, 5)); // q
console.log(myString.slice(-6, -1)); // Fox..
console.log(myString.slice(4)); // quick Red Fox...
console.log(myString.slice(-6)); // Fox...

split(), trim()

split() will split a string based on the supplied input and return an array. trim() will remove white space from the beginning and end of a string.

const myString = 'The quick Red Fox...';
console.log(myString.split(' ')); // [ "The", "quick", "Red", "Fox..." ]

const fruit = 'Apples || oranges || bananas';
console.log(fruit.split('||'));  // [ "Apples ", " oranges ", " bananas" ]

const paddedString = '      whitespace padded string      ';
console.log(paddedString.trim()) // whitespace padded string

includes(), match(), search(), test()

includes(str) returns true if the string contains str. search(), match(), and test() have similar purposes but accept regular expressions. We'll revisit these methods in the regular expression lesson.

const myString = 'The quick Red Fox...';
console.log(myString.includes('red')); // false
console.log(myString.includes('Red')); // true

replace(), replaceAll()

replace(str, newStr) and replaceAll(str, newStr) will replace the specified str in the given string, returning a new string. replace() replaces the first instance found, replaceAll() replaces ... all

const myString = 'The qqquick Red qqqFox...';
console.log(myString.replace('qqq', 'q')); // The quick Red qqqFox...
console.log(myString.replaceAll('qqq', '')); // The uick Red Fox...

charAt(idx), startsWith(str, [length]), endsWith(str, [length])

These are helpful for finding characters or strings at certain places in the string. They are effectively shortcuts for substring() or slice()

const myString = 'the lazy brown dog';
console.log(myString.charAt(6)); // z
console.log(myString.startsWith('the')); // true
console.log(myString.startsWith('lazy')); // false
console.log(myString.startsWith('lazy', 4)); // true
console.log(myString.endsWith('dog')); // true
console.log(myString.endsWith('lazy')); // false
console.log(myString.endsWith('lazy', 8)); // true
