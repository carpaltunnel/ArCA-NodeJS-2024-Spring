HTTP Servers in Node.js





Now that we know the high points of HTTP, let's build an HTTP server with Node.js.

http.createServer()

const http = require('http');

const server = http.createServer();
server.on('request', (request, response) => {
    console.log(`Got ${request.method} request for ${request.url}`);
    response.writeHead(200, {
    'Content-Type': 'text/plain'
    });

    response.end('Hello there');
});

server.listen(8080);
console.log('Server listening on port 8080...');

HTTP Server Events

  • checkContinue - Emitted whenever an HTTP Expect: 100-continue is received.
  • checkExpectation - Emitted whenever an HTTP Expect header is received and the value is not 100 Continue
  • clientError - Emitted when a client connection experiences an error.
  • close - Emitted when the server closes
  • connect - Emitted when a client issues an HTTP CONNECT method.
  • connection - Emitted when a new TCP stream is established
  • request - Emitted every time there is an HTTP request
  • upgrade - Emitted when an HTTP 1.1 Upgrade header is found

Request Properties

  • headers - The HTTP headers on the request
  • httpVersion - The HTTP version specified on the request
  • method - The HTTP method of the request
  • statusCode - The numeric HTTP status code
  • statusMessage - The short HTTP status message
  • url - The request URL

Response Properties / Methods

  • end() - Indicate the server can consider the request complete
  • getHeader(name) / getHeaders()
  • hasHeader(name)
  • removeHeader(name)
  • setHeader(name, value)
  • statusCode
  • statusMessage
  • write(chunk)
  • writeHead(statusCode, message, headers) - Send a response header
